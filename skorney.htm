Протестируйте пожалуйста контент на ссылках, расположенных ниже:
  • Исходная страница - блок не показывается при переходе по ссылке
  • Та же страница, только сохраненная на другом хосте - блок с показывается
  • Та же страница, только загружаемый внешний файл помещен на страницу - блок с показывается
  • Скрипт в - вызов за блоком - блок с показывается
  • Скрипт в - с явным контролем загрузки блока - блок с показывается
